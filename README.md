### demo 说明
在react中我们进行的都是对组件的操作，之前我们也学习了用props完成组件与组件之间的联系。但是props实现的是父组件向子组件的传递（react的单项数据流），但是很多时候我们想要实现的是子组件里的事件去改变父组件里的值，本例就实现这个功能。
